# Audio Files for Traind

Place royalty-free music tracks here with the following names:

## Required Tracks

1. **quiz-ambient.mp3** - Subtle electronic background music for gameplay (2-3 minute loop)
2. **quiz-tension.mp3** - Escalating tension music for final 30 seconds (30 second loop)
3. **victory-fanfare.mp3** - Celebration music for quiz completion (5-10 seconds)
4. **waiting-room.mp3** - Calm ambient music for pre-game lobby (2-3 minute loop)
5. **countdown.mp3** - Short countdown sound effect (3-5 seconds)

## Recommended Sources (Royalty-Free)

- [Pixabay Music](https://pixabay.com/music/)
- [Mixkit](https://mixkit.co/free-stock-music/)
- [freesound.org](https://freesound.org/)
- [Bensound](https://www.bensound.com/)

## Audio Specifications

- Format: MP3
- Bitrate: 128-192kbps (balance quality vs file size)
- Sample Rate: 44.1kHz
- Channels: Stereo or Mono

## Usage Notes

The music system will gracefully handle missing files - it will simply not play music if a track is unavailable. This allows the app to work without audio files during development.
